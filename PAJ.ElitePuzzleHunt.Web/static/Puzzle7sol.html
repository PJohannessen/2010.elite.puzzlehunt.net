Symbolic Cacaphony (SOLUTION)
Author: Cyberwrath
Difficulty: Hard

HINTS:
  1. How do these words "sound" together? The definitions and word length are not important.
  2. Beethoven often ate PIZZA in his MANOR.
  3. Knowing proper orchestral classification will be instrumental in solving this.
ANSWER: TIMPANI

Musical instruments can be found in each pair using consecutive letters from word 1 combined with consecutive letters from word 2. The title lets you form the word "symphony" from this same approach, and refers to the disguising of something relating to sound. Hint 1 alludes to this by putting "sound" in quotation marks as a double meaning. If you recall correctly, musical instruments can be broken down into 4 basic categories (woodwind, brass, percussion, string). Fill up the 4 columns with the instruments using the list's inherent ordering. Each category has a different amount of instruments, so you can figure out which column pertains to which classification without any guesswork. The number on each line is the index into each instrument.

From this, you can see it makes out BEDTIMESPANICKY (BEDTIMES / PANICKY). Using these two words, perform the same process from the beginning. Find the instrument!

The answer is TIMPANI.
  1. EMBASSY / DRAGOON = BASSOON (woodwind)
  2. SPECTRUM / PUPPETS = TRUMPET (brass)
  3. CRIBBAGE / PIPELESS = BAGPIPE (woodwind)
  4. SANGUINE / CUSTARD = GUITAR (string)
  5. RECYCLED / MISORDER = RECORDER (woodwind)
  6. CENTRIC / WRANGLER = TRIANGLE (percussion)
  7. ACRES / GROWTH = CRWTH (string)
  8. DECLARED / NINETEEN = CLARINET (woodwind)
  9. TUNIC / ABASH = TUBA (brass)
  10. REMARKS / ALPACAS = MARACAS (percussion)
  11. TURBOCAR / ORDINARY = OCARINA (woodwind)
  12. SEPPUKU / HOLELESS = UKULELE (string)
  13. BARBARIC / CAPSTONE = BARITONE (brass)
  14. PHARMACY / ICONICAL = HARMONICA (woodwind)
  15. BADLY / OGRES = LYRE (string)
