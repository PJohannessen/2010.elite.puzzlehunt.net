Une Façon de Cacher les Secrets Français (SOLUTION)
Author: Infil
Difficulty: Hard

HINTS:
  1. With modern technology, this "chiffre" is no longer "indechiffrable".
  2. After identifying the French figure the famous cipher is named after, apply a series of chain reactions to make progress.
  3. After decrypting each line, what should be done with your last unused key?
ANSWER: RIJNDAEL

The puzzle's title is French for "One way to hide French secrets". Well, one way to hide French secrets is to use a famous cryptographic method attributed to a French figure, the Vigenere cipher. This method was well-studied and appears in any serious discussion of the history of cryptography (including Wikipedia's), and is the only historically relevant method given a French name. It was also once known as "le chiffre indechiffrable", which means "the unbreakable cipher", which the first clue references (Googling the word "indechiffrable" brings up many pages on the Vigenere cipher).

The cipher encodes plaintext using another textual key by repeating the key until it's as long as the plaintext, then "adding up" (and wrapping around) the letters. More specifically, if A=0, B=1, ..., Z=25, then enciphering a letter x with a key letter y is simply x + y (mod 26). Deciphering is done by "subtracting" the letters. Your first key, as referenced by the preamble, is "middleoftheroad". Decipher the first line using this key; doing it by hand is pretty time consuming, so just use some web tool to do it. Take the answer from the first question and use it as the key to decipher the next line. The answers should fit in the blanks next to each question.

Repeat this process until you decipher all the lines. The answer to the final question is not the puzzle's solution, but rather another key to deciphering the first letters of all the answers (indicated by the boxes around the first blank on each row). This decrypts to the phrase ANSWERISRIJNDAEL. The answer to the puzzle is RIJNDAEL, which is the cipher currently used as the Advanced Encryption Standard, and effectively the most secure private-key encryption method ever developed.
